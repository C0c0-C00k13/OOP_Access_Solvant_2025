Calcul de la surface de protéine exposée au solvant.

Projet réalisé dans un environnement conda

- Data :
	- environment.yml : fichier YAML contenant les dependencies utilisées pour le calcul
	- Insuline.pdb : fichier PDB test utilisé pour calculer la sur 
 
- Res : Contient les résultats de calculs  

- Execution :
 	- Atome.py : script permettant de définir un objet atome ;
	- Point.py : script définissant le point d'un atome. Classe liée à Atome ;  
	- Main.py : fichier d'exécution. Réalise le calcul de surface ;
	
	- project.ipynb : notebook contenant l'ensemble du code pour une observation du script plsus détaillée
